
## Cline 무료 사용 가능 여부

**Cline 자체는 무료 오픈소스 VS Code 확장입니다.** 하지만 Cline이 동작하려면 **AI 모델 API가 필요**하며, 이 API 사용료가 발생할 수 있습니다.

### 💰 비용 구조

| 항목 | 비용 |
|------|------|
| Cline 확장 프로그램 | **무료** (오픈소스) |
| AI 모델 API 호출 | **유료** (사용량 기반) |

### 🔑 무료 또는 저렴하게 사용하는 방법

**1. Google Gemini API (무료 한도 있음)**
- Gemini 1.5 Flash / 2.0 Flash: **분당 15회 요청, 하루 1,500회 무료**
- Cline 설정에서 Google Gemini API를 선택하여 사용 가능
- [Google AI Studio](https://aistudio.google.com/)에서 API 키 무료 발급
- ⚠️ 무료 한도는 변경될 수 있음

**2. OpenRouter (일부 무료 모델)**
- [openrouter.ai](https://openrouter.ai)에서 무료 모델 제공 (예: Llama, Mistral 등)
- Cline에서 OpenRouter를 API Provider로 선택
- 무료 모델은 성능이 제한적일 수 있음

**3. Ollama (로컬 실행, 완전 무료)**
- 자신의 PC에 AI 모델을 설치하여 실행
- 인터넷 연결 불필요, API 비용 0원
- 단, 사양이 좋은 PC 필요 (최소 16GB RAM 권장, GPU 있으면 더 좋음)
- Cline에서 Ollama를 API Provider로 선택
- ⚠️ 로컬 모델은 Claude/GPT 대비 성능이 떨어질 수 있음

**4. Anthropic / OpenAI API (유료)**
- Claude 3.5 Sonnet, GPT-4o 등 최고 성능
- 사용량 기반 과금 (수천원~수만원/월 예상)
- 코딩 작업 품질은 가장 좋음

### 📌 추천 방법

| 상황 | 추천 |
|------|------|
| 완전 무료로 가볍게 테스트 | **Google Gemini API** (무료 한도) |
| 완전 무료, 오프라인 | **Ollama** (로컬 모델) |
| 최고 품질의 코딩 | **Anthropic Claude API** (유료) |

### ⚙️ Cline + Gemini API 설정 방법

1. VS Code에 Cline 확장 설치
2. [Google AI Studio](https://aistudio.google.com/) 접속 → API 키 발급 (무료)
3. Cline 설정 열기 (⚙️ 아이콘)
4. **API Provider** → **Gemini** 선택
5. 발급받은 API 키 입력
6. 모델 선택 (예: Gemini 2.0 Flash)

이렇게 하면 무료 한도 내에서 Cline을 사용할 수 있습니다.

### 💡 요약

- **Cline 자체는 무료**이지만 AI 모델 API 비용이 발생
- **Google Gemini API**를 사용하면 **무료 한도 내에서 사용 가능**
- 이 프롬프트로 `opic_study.html`을 만드는 정도의 작업은 무료 한도로 충분히 가능
- 더 복잡한 대규모 프로젝트라면 유료 API가 필요할 수 있음

  - 영어 블록이 나오면 pending_korean에서 꺼내서 zip으로 문장 쌍 생성
  - 기존 아이템에 문장 추가(extend) 또는 새 아이템 생성
- 문장이 없는 빈 아이템은 제거

[핵심 기능]

1. 학습 시퀀스 (startSequence):
   - 상태 "listening", "🔊 한국어 듣는 중..." 표시
   - speechSynthesis로 한국어 TTS (lang: 'ko-KR', rate: 0.9)
   - TTS 종료 후 0.5초 대기
   - 상태 "recording", "🔴 영어로 말하세요!" 표시
   - 음성 인식 대기 활성화 (isListeningForAnswer = true)
   - isTransitioning으로 중복 실행 방지

2. 음성 인식 (startApp):
   - SpeechRecognition / webkitSpeechRecognition 사용
   - lang: 'en-US', interimResults: true, continuous: true, maxAlternatives: 1
   - 미지원 시 "Chrome을 사용해주세요" 알림
   - 실시간 인식 텍스트 표시
   - final 결과 감지 시 checkAnswer() 호출
   - 음성 미감지 시 "🔇 음성이 감지되지 않았습니다." + 정답 보기 버튼
   - onend에서도 듣고 있으면 재시작 (try-catch)

3. 정답 확인 (checkAnswer):
   - Levenshtein Distance 알고리즘으로 유사도 계산
   - 정규화: 문장부호(.,!?;:'"()) 제거, 공백 정규화, 소문자 변환
   - 유사도 = 1 - (편집거리 / max(길이1, 길이2))
   - 80% 이상: 🎉 "훌륭해요!" (초록, correct)
   - 50% 이상: 🤔 "거의 맞았어요!" (주황, partial)
   - 50% 미만: 😅 "다시 해보세요!" (빨강, incorrect)
   - 유사도 퍼센트 표시, 정답 수 카운트 (answered 배열로 중복 방지)

4. 영어 미리보기 (togglePreview): 영어 답안 미리 보기 (토글)
5. 미리 듣기 (previewListen): 영어 답안 TTS (lang: 'en-US', rate: 0.85)
6. 정답 보기 (showAnswer): 정답 표시 + TTS 읽기
7. 자동 모드 (toggleAutoMode):
   - 토글 스위치 활성화/비활성화
   - 정답 후 자동 다음 이동 (80%↑: 3초, 50%↑: 5초, 미감지: 3초)
   - 카운트다운 "⏩ N초 후 다음으로..." 표시
   - 마지막 문장에서는 자동 이동 안 함
8. 네비게이션: 이전/다음 버튼 + 키보드(←, →, Esc)
9. 번역 없음 처리: english 빈 문자열 시 배지 표시, 미리보기/미리듣기 숨김

### 📝 사용 방법

1. **VS Code에 Cline 확장 설치** (Extensions에서 "Cline" 검색)
2. **Cline을 ACT 모드로 전환** (하단의 Plan/Act/Ask 토글에서 **Act** 선택)
3. **API Provider 설정** (Google Gemini API 무료 한도 추천)
4. **위 프롬프트를 Cline 채팅창에 붙여넣기**
5. Cline이 자동으로 다음을 수행:
   - `pip install python-pptx` 실행
   - `extract_all.py` 파일 작성 → 실행 (PPTX → JSON)
   - `generate_app.py` 파일 작성 → 실행 (JSON → HTML)
   - `opic_study.html` 생성 완료

> **장점**: Cline은 로컬 파일 시스템에 직접 접근하고 터미널 명령어를 실행할 수 있으므로, Gemini와 달리 사용자가 수동으로 스크립트를 실행할 필요 없이 **전 과정이 자동으로 진행**됩니다.
